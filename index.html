Quem sou eu?

Olá, nome é Paulo Henrique desenvolvedor iniciante que usa as tecnologias CSS, HTML, JS e Java. Tenho 22 anos e sou natural de Iguatu, no Ceará. Atualmente, estudo no Instituto Federal do Ceará, na área de TI. Amo jogos, músicas, documentários e aprender novas coisas. Atualmente, busco uma oportunidade no mercado de tecnologia. CSS é uma linguagem de estilos que define cores, fontes, tamanhos, posicionamento e qualquer outro valor estético para os elementos da página. HTML é uma linguagem de marcação que estrutura os elementos da página, como parágrafos, links, títulos, tabelas, imagens e até vídeos. JS é uma linguagem de programação que cria interação entre as páginas web e o usuário. Java é uma linguagem de programação orientada a objetos que roda em uma máquina virtual e pode ser usada para desenvolver aplicações web, desktop e móveis. Eu gosto de usar essas tecnologias porque elas me permitem criar sites dinâmicos, responsivos e atraentes. Eu também gosto de aprender novas coisas e me desafiar com novos projetos. Meu objetivo é me tornar um desenvolvedor profissional e contribuir para o avanço da tecnologia no meu país.
